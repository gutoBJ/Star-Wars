# 🌌 Star Wars Catalog

Um catálogo interativo de personagens, naves e planetas do universo Star Wars, desenvolvido com **HTML**, **CSS** e **JavaScript puro**, consumindo dados de APIs externas através de requisições HTTP.

## 🚀 Sobre o projeto

Este projeto foi desenvolvido com o objetivo de praticar conceitos fundamentais de desenvolvimento web, como:

- Consumo de APIs externas
- Requisições assíncronas com `fetch`
- Manipulação do DOM
- Paginação dinâmica
- Exibição de dados em modal
- Tratamento de erros
- Organização de código em JavaScript

Os dados são consumidos da API pública **SWAPI (Star Wars API)**, enquanto as imagens dos personagens são obtidas através de uma API complementar.

## 🛠 Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- API REST
- SWAPI

## ✨ Funcionalidades

- Listagem de personagens de Star Wars
- Paginação entre resultados
- Exibição de detalhes em modal
- Renderização dinâmica de imagens
- Conversão de altura e peso para melhor visualização
- Navegação intuitiva entre páginas

## 📚 Aprendizados

Durante o desenvolvimento deste projeto, pratiquei:

- Integração com APIs REST
- Uso de `async/await`
- Estruturação de funções assíncronas
- Manipulação de eventos
- Criação dinâmica de elementos HTML

## 🎯 Objetivo

O principal objetivo deste projeto foi consolidar conhecimentos em JavaScript puro e entender melhor como funciona a integração entre frontend e APIs externas.

## 📖 Créditos

Projeto desenvolvido durante o curso do Profissão Programador, com adaptações e melhorias implementadas durante o processo de aprendizado.
